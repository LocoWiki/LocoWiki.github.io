关于本站

这是一个用于展示 LeggedWiki 资源仓库的静态站点，使用 GitHub Pages（github.io）托管。

站点设计

  • 首页：按模块快速入口（规则 / 分享 / 论文 / 阅读清单）。
  • 文档页：通过浏览器实时拉取源仓库的 Markdown 并渲染成文档风格页面。
  • 下载页：提供整仓 ZIP、克隆命令、以及按模块获取的入口。

内容来源与版权

本站仅做“索引与展示”。仓库中收录的 PDF/PPT/外链等资料，其版权与署名权归原作者/原发布方所有。 如有侵权或不当公开，请在源仓库提交 Issue 以便及时处理。

常见问题

  • 文档加载失败？：文档页需要访问 raw.githubusercontent.com。若网络受限，请直接打开 GitHub 页面或使用镜像/代理。
  • 资源下载慢？：优先使用 git clone（或国内镜像），大文件建议启用 Git LFS。
