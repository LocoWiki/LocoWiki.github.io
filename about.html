Policy

内容来源与版权

本站仅做“索引与展示”。仓库中收录的 PDF/PPT/外链等资料，其版权与署名权归原作者/原发布方所有。 如有侵权或不当公开，请在源仓库提交 Issue 以便及时处理。

FAQ

常见问题

当前最容易遇到的问题主要来自网络访问、下载方式和旧飞书内嵌路径的历史兼容。

文档加载失败？

文档页需要访问 raw.githubusercontent.com。若网络受限，请直接打开 GitHub 页面或使用镜像/代理。

资源下载慢？

优先使用 git clone（或国内镜像），大文件建议启用 Git LFS。

为什么不再强调飞书内嵌？

因为第三方内嵌受登录态、跨域策略和浏览器限制影响太大，现在统一以同步后的仓库文档作为站内阅读入口。
