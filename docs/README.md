# 开发文档（docs）

本目录用于存放站点维护与开发文档，避免在前台页面中堆积面向维护者的提示信息。

## 文档索引

- `feishu-sync.md`：飞书知识库同步链路（GitHub Actions / relay / 状态文件）
- `frontend-copy-guidelines.md`：前端页面文案规范与命名约定

## 使用原则

- 面向访客的信息放在前台页面（`index.html/about.html/downloads.html/docs.html?path=__feishu_wiki__`）
- 面向维护者的操作说明统一放到本目录
- 根目录 `README.md` 只保留项目总览与入口，技术细节通过链接跳转到本目录
